cgroup工作原理
=============

		-------------         -------------                 --------------
	   |     cpu     |       |    memory   |               |    net_cls   |
	    -------------         -------------                 --------------
	          |                     |                              |       
	          |                     |                              |      
	    ------------------------------------        -------------------------------
	   |          --------------            |      |        ------------           |
	   |         | cpu_mem_cgrs |           |      |       |  net_cgrs  |          |
	   |          --------------            |      |        ------------           |
	   |                |                   |      |           |         --------  |
	   |       --------------------         |      |           |--------|  cgr3  | |
	   |      |                    |        |      |           |         --------  |
	   |  ---------           -----------   |      |           |    --------  |    |
	   | |  cgr1   |         |   cgr2    |  |      |           |---|  cgr4  | |    |
	   |  ---------           -----------   |      |                --------  |    |
	   |      |                    |        |      |                    |     |    |      
	    ------|--------------------|--------        --------------------|-----|----
	          |                    |                                    |     |
	          |                    |                                    |     |
	          |                    |                                    |     |
	        ----------------------------------------------------------------------
	       |                                                                      |
	       |                         对应关系（N : N）                              |
	       |                                                                      |
	        ----------------------------------------------------------------------
	                       |                                |
	                       |                                |
	                  -----------                      -----------
	                 |  css_set1 |                    |  css_set2 |
	                  -----------                      -----------
	                       |                                |
	               -----------------               -------------------
	              |        |        |             |         |         |
	              P        P        P             P         P         P
	              
* 一个进程（P）只能隶属于一个css_set，一个css_set可以包含多个进程。隶属于同一css_set的进程受到同一个css_set所关联的资源限制。
* css_set与cgroup是 n:n 的对应关系。`但是css_net不能关联同一层级下的不同cgroup`。比如css_set1可以同时关联cgr1和cgr3，但是不能同时关联cgr1和cgr2。
